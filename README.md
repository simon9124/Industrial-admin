## Industrial-admin

**平台介绍：**

- 基于 view-admin 和 工业互联网 4.0 的质量追溯平台
- 平台用途：智能检测生产产品，严格制定标准、极大提高效率
- 适用场所：工厂、车间等

*与实际生产中的真实业务逻辑相符，暂未连数据库，数据为前端mock

**平台架构：**

nodejs + vuejs + iview + echart

**账户密码：**

- admin：管理员
- factor：工程师
- workshop_manager：车间主管
- proline_leader：产线线长

*密码均为2019@cestc

**安装教程：**

cd ~

npm install
npm run dev

修改 vue.config.js
npm run build
